List Lager

Ingredients

  • A location for the list, vertically orientated
  • Subject or topic for the list
  • A type of list that fits its topic and purpose
    • Unordered List <ul></ul>
    • Ordered list <ol><ol>
  • Items to present in the list, topic related

Instructions

  1. Place the opening tag of the list at the chosen location
  2. Drop list tags <li></li> for each item to be showed in the list. Use a new rule for every item.
  3. Write the to be viewed content in between the openinng and closing list tags
