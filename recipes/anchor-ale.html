Anchor Ale

Ingredients:

  • Anchor opening tag: <a>
  • Anchor closing tag: </a>
  • HyperText Reference attribute: href=""
  • Reference value:
    • Absolute link (remote)
    • Relative link (local)
    • Root-relative (root)
  • Content to viewed

Instructions

  1. Pick a location for the anchor element to be viewed
  2. Drop the anchor opening tag (<a>)
  3. Within the brackets but after the "a", place the HyperText Reference attribute (href="")
  4. Nest the chosen reference value in between the double quotes
  5. Note down the to be viewed content after the closing bracket (>) of the anchor opening tag
  6. Finish the element with the closing tag (</a>)
