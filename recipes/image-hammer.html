Image Hammer

Ingredients

  • Image tag (<img>)
  • Much visited space for the image to be viewed
  • Image source attribute (src="")
  • An actual source for the image, local (/example-directory/example.jpg) or remote (https://www.example.com/example-image)
  • Alternative text attribute (alt="")
  • Input for alternative attribute

Instructions

  1. Choose a proper location to fit the image element in
  2. Place the self-closing - also called "empty" or "void" - image tag (<img>)
  3. Within the brackets but after the text, place a "source" attribute (src-"")
  4. Provide a source to make the <img> tag functional, place it in between the double quotes
  5. Place the "alternative text" attribute within the image tag
  6. Write a dense descriptive text about the image in between the double quotes of the alt attribute
